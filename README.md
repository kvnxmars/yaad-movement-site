# YAAD MOVEMENT Website

This repository contains the main static website for the YAAD MOVEMENT landing page. The site is centered around the single-page experience defined in `code-space/index.html`.

## What the site is about

YAAD MOVEMENT is an event organizing platform focused on live music performances and artist promotion. The website highlights past and future events, the team behind the movement, music categories, services offered, and contact details for bookings and inquiries.

## Technologies used

- HTML5 for the page structure
- CSS for styling and responsive layout
- JavaScript for interactive elements such as the mobile menu, scroll-to-top button, fade-in animations, and contact form handling
- Font Awesome for social and navigation icons
- Google Fonts for typography

## Directory of the main page

- `code-space/index.html` — primary static landing page for the YAAD MOVEMENT site
